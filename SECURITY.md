# Security policy

Only Katrina Li, through the verified `katlicolumbia-a11y` owner accounts, is authorized to approve publication changes to this project.

Do not disclose suspected credential exposure publicly. Report unauthorized changes or impersonation privately to `katrina.li.columbia@gmail.com` and preserve screenshots, URLs, timestamps, commit identifiers, and email headers as evidence.

The `main` branch is protected against direct pushes, force-pushes, and deletion. Publication requires the repository integrity pipeline to pass.
